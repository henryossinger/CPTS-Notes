# Notes repository for the CPTS 

This repository acts as a centralized location for my CPTS notes. I enjoy using Obsidian as my primary notes app, which to sync across multiple devices requires a subscription. To get past this, I have setup a git repository on my local machine, and push the changes to this repository after each studying session. 

Doing so allows me to use the same notes across multiple devices without having to pay for the subscription. Obsidian uses directories and markdown files for the actual notes, so making this work across multiple platforms is copy and paste.
